# 1000 баллов — Telegram WebApp / GitHub Pages

Эта версия специально сделана БЕЗ папок.
Все файлы должны лежать в корне репозитория рядом с index.html:

- index.html
- styles.css
- app.js
- manifest.json
- part-001.json ... part-050.json
- generate_meaningful_base.py
- README.md

## Как загрузить на GitHub через браузер

1. Открой репозиторий.
2. Нажми Add file → Upload files.
3. Выдели ВСЕ файлы из распакованного ZIP, не папку.
4. Загрузи файлы.
5. В Settings → Pages выбери Branch: main, Folder: /root.

Важно: `index.html`, `manifest.json` и все `part-*.json` должны быть рядом в одном месте.
